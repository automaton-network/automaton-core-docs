void automaton::core::network::simulated_connection::async_read ( std::shared_ptr< char >  buffer,
uint32_t  buffer_size,
uint32_t  num_bytes,
uint32_t  id 
)
virtual

Function that is used to read incoming messages. Messages could be received but not read if this function was not called. When a message is ready to be read, handler's on_message_received will be called. If an error occures, handler's on_connection_error will be called.

See also
connection_handler
Parameters
[in]bufferreceived message with size max(buffer_size, num_bytes) will be stored here
[in]buffer_sizethe size of the buffer
[in]num_bytesthe number of bytes that are required to be read. If a message with greater size is received, only num_bytes will be read. If a message with smaller size is received, it will not be read until at least num_bytes are received. If num_bytes = 0 then a message will be read when it is received no matter the size (but max buffer_size)
[in]idgiven by the user to identify this specific read call. It will be used from the handler when on_message_received is called.
See also
connection_handler::on_message_received

This function does not create event. Actual reading will happen when the other endpoint sends a message and it arrives (send event is handled and read event is created).
