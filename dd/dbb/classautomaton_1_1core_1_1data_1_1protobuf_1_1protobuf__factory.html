◆ get_field_type()

string automaton::core::data::protobuf::protobuf_factory::get_field_type ( uint32_t  schema_id,
uint32_t  tag 
) const
virtual

Returns the type (as a string) of the field with the given tag in the schema with the given id. If the given schema id is not valid or there is no field with the given tag, exception will be thrown.

If the type is message, returns 'message'. To get the type of a message field use get_message_field_type().

If the type is enum, returns 'enum'. To get the type of a enum field use get_enum_field_type().

Use the result of this function in set_*() and get_*() and in set_repeated_*() and get_repeated_*() if the field is repeated.
